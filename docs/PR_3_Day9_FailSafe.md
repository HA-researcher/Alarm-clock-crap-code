##  Update Summary / 変更概要
- [APIエラー時（課題取得失敗・AIレビュー失敗）に自動で緊急停止処理（フェールセーフ）が発動するよう実装しました。]

##  Details / 詳細
- [app/challenge/page.tsx 内の `fetchChallenge`（課題取得APIの呼び出し）および `submitCode`（コードレビューAPIの呼び出し）処理において、エラー発生時（catch ブロック）に手動でボタンを押さなくとも自動で `handleEmergencyStop` をトリガーするように改修しました。]
- [これにより、APIキーの制限超過などのエラー発生時には専用の「後輩ちゃんのクソコードは〜」UIに移行し、強制的にアラームを停止させることができるようになります。]
